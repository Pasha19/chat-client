# SSE chat client

## Requirements

- node.js
- yarn

## Install

- clone sources
- `yarn install`

## Configure

- copy `app.config.js.dist` to `app.config.js`
- put `host` and `port` for dev server, and `api` for backend

## Run

`yarn run build` then copy content of `dist` directory to web server root or `yarn run server` to use dev server
